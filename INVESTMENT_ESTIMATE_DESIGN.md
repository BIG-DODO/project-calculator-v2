# 投资估算表设计文档

## 一、设计目标

根据产品配置完成后的指标数据，自动计算项目投资估算，形成：
- **页面端展示**：简化版投资估算表，由完整版汇总反推。
- **下载表格**：完整版 Excel 表格，带公式、可编辑。

核心原则：**简化版必须由完整版数据汇总而来，不能反向推导。**

---

## 二、计算流程

```
┌─────────────────────────────────────────────────────────────┐
│  输入：产品配置结果 + 基础参数 + 用户手动输入项              │
│  （总建筑面积、地上建筑面积、地下建筑面积、总用地、绿地率、  │
│   建筑密度、各产品面积、产业大厦高度、城市、土地出让金等）  │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  步骤 1：计算完整版各小项成本                                │
│  完整版小项成本 = 表格参考单价 × 工程量                      │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  步骤 2：汇总完整版一级/二级科目                             │
│  并计算增值税、不可预见费、利息支出                          │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  步骤 3：由完整版反推简化版综合单价                          │
│  综合单价 = 一级/二级科目总成本 ÷ 对应面积指标               │
└─────────────────────────────────────────────────────────────┘
                              ↓
        页面展示简化版                  下载完整版 Excel（带公式）
```

## 三、完整版 Excel 输出结构

完整版包含以下 sheet：

| Sheet 名称 | 说明 |
|------------|------|
| 规划指标 | 产品配置完成后的核心规划指标，作为后续工程量的引用源 |
| 加权平均造价表 | 轻钢、分栋、分层、产业大厦的单价及加权计算 |
| 投资估算完整版 | 完整的投资估算科目及成本明细 |

### 3.1 输出字段

| 字段 | 说明 | Excel 单元格示例 |
|------|------|------------------|
| 科目编码 | 一、1-1、3-4-2-1 等层级编码 | A 列 |
| 成本科目 | 一级/二级/三级科目名称 | B 列 |
| 指标单位 | 元/㎡、万元/亩、元/m、KVA、项、比例等 | C 列 |
| 成本指标 | 单价或费率 | D 列 |
| 工程量 | 面积、长度、个数、KVA 等 | E 列 |
| 成本（万元） | 单项成本 = D × E ÷ 10000 | F 列（带公式） |
| 单位建面成本（元/㎡） | F ÷ 各自工程量（单体建安/地下室/配套楼/宿舍）或总建筑面积（公摊类） | G 列（带公式） |
| 单位地上建面成本（元/㎡） | F ÷ 各自地上工程量或地上总建筑面积；地下室为 0 | H 列（带公式） |
| 科目说明 | 计算口径说明 | I 列 |

### 6.5 财务费用

```
总投资（计息基数） = 土地配套费用 + 前期费用 + 建安工程成本
总建设期 = 开发期数 × 单期开发周期
平均计息年限 = 总建设期 / 2
利息支出 = 总投资 × 融资占比 × 融资利率 × 平均计息年限
银行费用支出 = 利息支出 > 0 ? 30 万元 : 0
```

## 七、数字精度规则

- 表格中所有参考单价默认保留 **两位小数**。
- 完整版输出：
  - 成本指标：保留两位小数。
  - 工程量：保留两位小数。
  - 成本（万元）：保留两位小数。
  - 单位建面成本、单位地上建面成本：保留两位小数。
- 简化版输出：
  - 综合单价：保留两位小数。
  - 成本（万元）：保留两位小数。
  - 单位建面成本、单位地上建面成本：保留两位小数。
  - 占比：保留两位小数（显示为百分比）。

## 八、Excel 下载要求

1. **完整版表格**：复刻真实项目「7-成本汇总表」结构，包含所有一级/二级/三级科目。
2. **新增基础数据 sheet**：
   - **规划指标 sheet**：写入产品配置后的核心指标，作为工程量引用源。
   - **加权平均造价表 sheet**：计算各产品加权平均单价，供单体建安成本引用。
3. **带公式可编辑**：
   - 成本（万元）列使用 Excel 公式：`D × E / 10000`。
   - 单位建面成本列：公摊类使用 `F / 总建筑面积`；单体建安、地下室、配套楼/宿舍使用 `F / 各自工程量`。
   - 单位地上建面成本列：公摊类使用 `F / 地上总建筑面积`；地下室为 0；配套楼/宿舍及单体建安使用各自地上工程量。
   - 汇总行使用 `SUM` 公式。
   - 不可预见费使用公式：`（土地+前期+建安）× 3%`。
   - 增值税使用公式：土地 6%、前期 6%、建安 9%。
4. **用户可编辑区域**：
   - 土地出让金单价、大市政配套费单价。
   - 各产品单方造价（轻钢、分栋独栋/双拼、分层独栋/双拼、产业大厦分档）。
   - 融资利率、融资占比、开发期数、单期开发周期。
   - 其他可调整单价（如景观、基础设施小项单价）。
5. **自动联动**：修改输入单价后，成本、单位成本、汇总行自动重算。

## 九、与产品配置的接口数据

从 product-config-v2.html 可获取以下数据：

| 数据项 | 来源 |
|--------|------|
| 总用地面积 | 基础参数输入 |

| 配套占比 | 基础参数输入 |
| 研发办公占比 | 基础参数输入 |

| 配套宿舍建筑面积 | 产品配置结果 |
