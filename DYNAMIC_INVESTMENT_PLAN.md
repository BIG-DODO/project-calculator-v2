## 一、JS 计算模块（`financial-modules.js`）

### 1.1 函数签名与前置校验

`NS.calculateDynamicAnalysis(inputs, result, projectData, staticAnalysis)` 保持现有签名不变。

页面端在调用前校验：

- `window.investmentEstimateResult` 缺失 → 提示「请先进行投资估算」；
- `window.staticAnalysisResult` 缺失 → 提示「请先进行静态投资分析」。

投资估算结果单独取（财务费用、融资参数需要），静态结果承载面积与价格口径。

### 1.2 输入解析（默认值）

| 输入项 | 默认值 |
|--------|--------|
| 折现率 | 8% |
| 运营期 | 20 年 |
| 租金年增长率 | 2% |
| 售价年增长率 | 0% |
| 销售去化速度 | 1.5 万㎡/年（选项 1.0/1.2/1.5/1.8/2.0/2.5/3.0） |
| 租赁去化速度 | 1.5 万㎡/年（选项同上） |
| 稳定出租率 | 90%（取静态分析输入值） |
| 预售开关 | 关闭 |
| 建设期第 1/2 年去化比例 | 0% / 30% |

### 1.3 基础量来源

- 总投资、融资利率、融资占比、财务费用 ← `investmentEstimateResult`；
- 可售面积 `soldAreaTotal`、可租面积 `rentedAreaTotal`、加权平均售价（raw 精度）、加权平均日租金（raw 精度）、营销/管理/运营费率、土增税扣除项目合计、销售各项成本 ← `staticAnalysisResult`；
- 建设期年数 = 开发期数 × 单期开发周期（投资估算输入，默认 2 年）。

### 1.4 去化引擎

生成两个逐年数组 `saleDeplete[t]`、`rentDeplete[t]`（t = 0 .. 建设期年数+运营期年数−1）：

- 销售上限 = 可售面积（100%）；租赁上限 = 可租面积 × 稳定出租率（满租口径）；
- 开预售：建设期第 1 年 = 比例₁ × 总量，第 2 年 = 比例₂ × 总量（销售、租赁同比例，租赁为建设期预租）；
- 之后每年 = `MIN(去化速度, 上限 − 累计已去化)`，直到达到上限。

### 1.5 逐年现金流（三阶段）

每年计算一行（金额单位：万元）：

- **销售回款** = 当年销售去化 × 加权售价 × (1+售价增长率)^t ÷ 10000；
- **租金收入** = 当年末累计已租面积（含预租，建设期不计）× 日租金 × 360 × (1+租金增长率)^t ÷ 10000；
- **销售税费** = 税金及附加（回款/1.09×0.006）+ 营销费 + 管理费（均随当年回款，口径同静态）；
- **运营税费** = 租金×0.006 + 租金×12% + 累计已租面积×6 元 ÷ 10000；
- **运营成本** = 租金 × 运营费率；
- **可用资金** = 销售回款 + 租金收入 − 销售税费 − 运营税费 − 运营成本；
- **清算**：销售去化完成当年，以动态累计销售回款为转让收入、扣除项目沿用静态口径重算四级累进土增税；所得税 = max(0, 动态利润总额) × 25%（利润总额 = 动态累计回款 − 静态土地/建安/费用口径）；从该年可用资金中一次性扣除；
- **建设期**：总投资支出 = 总投资/建设期年数；自有出资 = 总投资×(1−融资占比)/建设期年数；贷款提款 = max(0, 投资支出 − 自有出资 − 当年预售回款)；建设期利息不重复计（已含在财务费用内）；
- **运营期**：利息 = 剩余本金 × 融资利率；还本 = min(可用资金 − 利息, 剩余本金)；股东可分配 = 可用资金 − 利息 − 还本（为负记股东补足，`equityShortfall=true` 供页面预警）；
- **项目净现金流**（全投资口径）= 可用资金（建设期再减总投资支出）；
- **自有资金现金流** = 建设期 −自有出资 / 运营期 股东可分配。

### 1.6 指标与敏感性

- NPV = Σ 项目净现金流 / (1+r)^t；
- IRR：对项目净现金流求解；自有资金 IRR：对自有资金现金流求解；均用二分法（−99% ~ 1000%，100 次迭代，无变号返回 null，显示「—」）；
- 动态投资回收期 = 项目累计折现现金流由负转正的年份，线性插值到 0.1 年；
- 敏感性：6 变量（售价、租金、建安成本、土地价格、出租率、去化周期）× ±10%/±20% 共 24 组，每组扰动后整体重跑模型（建安/土地扰动调整总投资及成本口径，出租率调整满租上限，去化周期按去化速度反向换算），输出 NPV / IRR / 自有资金 IRR / 动态回收期。

## 二、页面端（`product-config-v2.html`）

- `renderDynamicAnalysis()`：输入卡片（基础假设 / 去化设置 / 预售设置默认折叠 / 融资参数只读展示）+ 计算按钮 + 前置校验提示；
- `renderDynamicAnalysisResult()`：
  - 4 个指标卡：NPV、IRR、自有资金 IRR、动态回收期；
  - 关键年份现金流表（t = 0/1/2/5/10/20，股东补足年份标红预警）；
  - 累计净现金流曲线（复用项目现有图表方案）；
  - 敏感性表 + 龙卷风图（仅售价/租金/建安 3 组，按影响幅度排序）；
- `downloadDynamicAnalysis()`：校验后调用 Excel 导出。

## 三、Excel 导出（`financial-modules.js` 新增 `NS.downloadDynamicAnalysisExcel`）

### Sheet 1：多年现金流表（按年全量）

- 顶部**参数区**：总投资、自有出资、贷款利率、加权售价/租金、增长率、去化速度、可售/可租面积、出租率、各费率、清算税额、建设期年数、预售比例——全部可编辑输入格，预填 JS 缓存值；
- 下方按年全量行（t=0 起，按运营期年数预生成）：
  - 去化面积用 `MIN(速度, 上限−累计)` 公式链；
  - 回款/收入/税费/利息/还本/股东可分配/净现金流/折现/累计均写公式引用参数区；
- NPV/IRR 行用 Excel 内置 `NPV()`/`IRR()` 函数，同时写入 JS 预计算缓存值；
- 沿用 `cell()`/`moneyCell()` 工具函数与 `ROUND(...,2)` 约定，数字单元格显式 `t='n'`；
- 结构性变化（预售开关、运营期年数）通过预生成最大行数 + IF 判断显示 0 规避。

### Sheet 2：敏感性分析

6 变量 × 4 档全量表（缓存值 + 基准列）。

### Sheet 3：关键指标汇总

NPV、IRR、自有资金 IRR、动态回收期、建设期/销售完成年/满租年等关键节点。
