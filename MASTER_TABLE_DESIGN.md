# 总表集成设计文档

## 一、设计目标

将项目全周期涉及的各类表格集成到一个 Excel 总表中，实现数据单向传递和公式联动，方便用户下载后在本地继续编辑和测算。

---

## 二、总表结构

总表分为 **四大部分**，共包含多个 sheet：

| 部分 | 说明 | 包含 Sheet |
|------|------|-----------|
| 第一部分：三表联动 | 规划指标、产品配置、估算结果的基础联动 | 5 个 sheet |
| 第二部分：投资估算 | 基于产品配置结果的成本测算 | 4 个 sheet |
| 第三部分：静态投资分析 | 基于投资估算和租售价格的收益测算 | 5 个 sheet |
| 第四部分：动态投资分析 | 基于静态分析的多年现金流和收益指标（待开发） | 3 个占位 sheet |

---

## 三、第一部分：三表联动（5 个 Sheet）

| Sheet | 名称 | 内容 | 联动关系 |
|-------|------|------|----------|
| Sheet 1 | 规划指标初始值 | 用户输入的原始规划指标 | 用户输入源 |
| Sheet 2 | 产品配置选择 | 用户勾选的产品类型、层数、面积段 | 产品配置算法输入（只读展示） |
| Sheet 3 | 指标估算 | 基于初始规划指标估算的综合指标 | ← 与 Sheet 1 深度联动 |
| Sheet 4 | 产品配置详表 | 产品配置算法输出的详细配置 | 内部公式联动 |
| Sheet 5 | 总体经济技术指标表 | 基于产品配置详表的最终指标 | ← 与 Sheet 4 深度联动 |

### Sheet 1 规划指标初始值字段

| 字段 | 单位 | 说明 |
|------|------|------|
| 规划用地面积 | ㎡ | 用户输入 |
| 容积率 | — | 用户输入 |
| 建筑密度上限 | % | 根据容积率自动计算或用户输入 |
| 绿地率 | % | 用户输入 |
| 限高 | m | 用户输入 |
| 研发办公用房占比 | % | 用户输入 |
| 配套用房占比 | % | 用户输入 |
| 厂房配建指标 | 辆/100㎡ | 用户输入 |
| 配套用房配建指标 | 辆/100㎡ | 用户输入 |
| 城市/区域 | — | 用于默认参数判断（如上海默认无大市政配套费） |

### Sheet 3 指标估算字段

与当前指标估算表一致，所有字段公式引用 Sheet 1。

### Sheet 4 产品配置详表字段

与当前产品配置详表一致，字段间通过公式联动：
- 修改首层层高 → 自动更新总层高。
- 修改层数 → 自动更新计容面积、建筑高度。
- 修改面积段/栋数 → 自动更新基底面积、计容面积、总建筑面积。
- 修改容积率不会触发产品配置算法重算。
- 租金、售价字段按产品类型填入默认值，用户可编辑。

### Sheet 5 总体经济技术指标表字段

与当前总体技术经济指标表一致，所有字段引用 Sheet 4。

## 四、第二部分：投资估算（4 个 Sheet）

参考 `INVESTMENT_ESTIMATE_DESIGN.md`，包含：

| Sheet | 内容 |
|-------|------|
| 规划指标 | 为完整版投资估算提供工程量引用源，字段与投资估算模块中的「规划指标」sheet 一致 |
| 加权平均造价表 | 按产品类型及形态列出面积、单价、成本、加权平均单价，供单体建安单价引用 |
| 完整版投资估算 | 完整成本汇总表结构 |
| 简化版投资估算 | 由完整版汇总反推 |

### 数据来源

投资估算的所有基础数据来自 **Sheet 5 总体经济技术指标表** 和 **投资估算内「规划指标」sheet**，包括：
- 总建筑面积
- 地上总建筑面积
- 地下总建筑面积
- 总用地面积
- 各产品建筑面积（轻钢、分栋、分层、产业大厦、配套楼、配套宿舍）
- 分栋/分层的独栋、双拼/三拼建筑面积
- 产业大厦建筑高度
- 配套楼面积、配套宿舍面积
- 道路面积、建筑密度、绿地率等

用户输入项包括：
- 土地出让金单价
- 大市政配套费单价
- 海绵城市是否启用
- 各产品单方造价（轻钢 1500、分栋 2300/2200、分层 2400/2300、产业大厦按高度分档等）
- 融资利率（默认 5%）
- 融资占比
- 开发期数、单期开发周期

### 关键计算规则

- 财务费用 = (土地配套费用 + 前期费用 + 建安工程成本) × 融资占比 × 融资利率 × (开发期数 × 单期周期 / 2) + 银行费用
- 银行费用默认 30 万元，仅当利息支出 > 0 时计入。
- 上海默认无大市政配套费。
- 所有成本小计通过 `ROUND(SUM(...),2)` 计算。
- 增值税采用 `D*E` 联动结构：
  - 土地配套增值税：E = 含税工程量 / 1.06，F = 0.06 × E
  - 前期费用增值税：E = ROUND(前期小计,2) / 1.06，F = 0.06 × E
  - 建安增值税：D = 0.09，E = ROUND(建安小计,2) / 1.09，F = D × E

## 五、第三部分：静态投资分析（5 个 Sheet）

参考 `STATIC_INVESTMENT_DESIGN.md`，包含：

| Sheet | 内容 |
|-------|------|
| 销售测算 | 规划指标、建造成本、销售测算结果 |
| 租赁测算 | 规划指标、建造成本、租赁测算结果 |
| 租售面积分配 | 各产品大类销售/租赁面积、单价、加权平均价格 |
| 土地增值税测算表 | 四级超率累进测算 |
| 综合汇总 | 总投资、利润、NOI、回收期等核心指标 |

### 数据来源

静态投资分析的基础数据来自 **Sheet 5 总体经济技术指标表**、**投资估算结果** 和 **用户输入的租售价格**，包括：
- 总建筑面积、地上建面、地下建面、计容建面
- 各产品建筑面积
- 总投资、综合单方成本、利息支出
- 配套占比、研发办公占比、配套楼面积
- 可分割销售比例、分栋/分层租金和售价、出租率（默认 90%）等

### 关键计算规则

- 可分割销售面积 = 计容总建面 × (1 - 配套占比 - 研发办公占比) × 可分割销售比例
- 可租面积 = 地上总建面 - 可分割销售面积 - 配套楼面积
- 销售优先级：轻钢 → 分栋 → 分层 → 产业大厦
- 租赁优先级：产业大厦 → 分层 → 分栋 → 轻钢；配套宿舍强制全部出租
- 销售收入 = Σ(各产品销售面积 × 售价)，引用「租售面积分配」合计
- 年租金收入口径：日租金 × 360
- 有效年租金 = 年租金收入 × 出租率 / 100
- 净租赁收入 per sqm = 有效年租金 - 税金及附加 - 房产税 - 土地使用税 - 运营费用
- 净租赁收入总额 = 净租赁收入 per sqm × 可租面积 / 10000
- 租赁总投 = 可租面积 × 综合单方成本 / 10000 + 分摊财务费用
- 静态投资回收期 = 总投资 / 租赁年净收入
- 资金盈余/缺口 = 总投资 - (1 - 融资占比) × 总投资 - 销售收入
- 加权平均售价/租金在 Excel 中保留 6 位小数精度，显示两位

## 七、数据流向与联动规则

### 7.1 单向传递原则

```
产品配置详表（Sheet 4）
        ↓
总体经济技术指标表（Sheet 5）
        ↓
├─→ 投资估算（规划指标 → 加权平均造价表 → 完整版 → 简化版）
├─→ 静态投资分析
└─→ 动态投资分析（待开发）
```

### 7.2 联动规则

| 修改位置 | 影响范围 | 是否反向影响 |
|----------|----------|--------------|
| Sheet 1 规划指标初始值 | Sheet 3 指标估算 | 不影响 Sheet 4/5 及财务测算 |
| Sheet 4 产品配置详表 | Sheet 5 总体经济技术指标表 → 投资/静态/动态 | 不反向触发产品配置算法 |
| Sheet 5 总体经济技术指标表 | 投资/静态/动态 | 不反向改变 Sheet 4 |
| 投资估算参数 | 只影响自身成本计算结果 | 不影响 Sheet 4/5 |
| 静态分析参数 | 只影响自身收益计算结果 | 不影响 Sheet 4/5 |
| 动态分析参数 | 只影响自身现金流/收益指标 | 不影响 Sheet 4/5 |

### 7.3 不联动

- 投资估算、静态分析、动态分析 **不与 Sheet 3 指标估算表联动**。
- 财务测算参数修改不会反向影响最初的规划指标估算，只基于产品配置和总体经济指标进行自身计算。

## 九、Excel 下载要求

1. **总表为一个 Excel 文件**，包含上述所有 sheet。
2. **所有数值引用基础数据单元格**，支持用户修改后自动重算。
3. **Sheet 4 产品配置详表内部公式联动**，修改单值后相关字段自动更新。
4. **Sheet 5 及以后的表格引用 Sheet 4 和 Sheet 5**，确保数据一致性。
5. **Sheet 1 和 Sheet 3 独立联动**，不影响后续财务测算 sheet。
6. 所有数字单元格显式设置类型，避免字符串类型导致 `#VALUE!`。
7. 所有求和公式包 `ROUND(...,2)`，中间结果也包 `ROUND(...,2)`。

---

## 十、数字精度

- 面积：保留两位小数。
- 单价：保留两位小数。
- 金额（万元）：保留两位小数。
- 比率：保留两位小数（显示为百分比）。
- 加权平均售价/租金：Excel 公式保留 6 位小数，显示两位。
