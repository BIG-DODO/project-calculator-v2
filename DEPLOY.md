## 部署方式：GitHub Pages（推荐，完全免费）

### 步骤一：注册/登录 GitHub

1. 打开 [https://github.com](https://github.com)
2. 有账号直接登录，没有就注册（免费，1分钟搞定）

### 步骤二：创建仓库

1. 登录后，点击右上角 **+** 按钮 → **New repository**
2. **Repository name** 填写：`project-calculator`（或你喜欢的名字）
3. 下面保持默认（Public / 不勾选 Add a README）
4. 点击最下方 **Create repository**

### 步骤三：上传文件

**方法一：直接拖拽（最简单）**

1. 在仓库页面，点击 **Add file** → **Upload files**
2. 把 `index.html` 文件从电脑文件夹里拖进去
3. 页面下方点击 **Commit changes**

**方法二：命令行（如果你熟悉 Git）**

```bash
cd project-calculator
git init
git add index.html
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/project-calculator.git
git push -u origin main
```

### 步骤四：开启 GitHub Pages

1. 在仓库页面，点击顶部的 **Settings**（设置页签）
2. 左侧菜单找到 **Pages**（在底部）
3. **Source** 选择 **Deploy from a branch**
4. 分支选 **main**，文件夹选 **/ (root)**
5. 点击 **Save**
6. 等 1-2 分钟，页面上方会出现一个绿色框，显示类似：
   ```
   Your site is live at https://你的用户名.github.io/project-calculator/
   ```

### 步骤五：分享给团队

1. 把上面那个绿色链接复制下来
2. 发到微信/钉钉群
3. 同事点开链接，就能在手机上直接使用了

## 常见问题

**Q：GitHub Pages 打开慢怎么办？**
A：GitHub 在国内访问可能慢，可以用 Vercel 或腾讯云静态托管加速，或者让同事使用 Wi-Fi 环境下打开。

**Q：可以设置密码访问吗？**
A：GitHub Pages 不支持密码，但链接只有知道的人才能访问。如果需要更高安全性，可以部署到内网服务器或企业微信自建应用。

**Q：手机浏览器打开后数据会保存吗？**
A：目前版本纯前端计算，不保存历史。后续如需保存历史记录，可以加入 localStorage 功能。

**Q：如果我不想让别人看到怎么办？**
A：GitHub 仓库设置为 Private（私有），但 GitHub Pages 对免费用户只支持 Public 仓库。如需私有部署，推荐 Vercel 或腾讯云静态托管。
