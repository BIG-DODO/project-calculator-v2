# 规划/产品总表下载（三表联动）需求文档

## 一、目标

将「指标估算」「产品配置」「总体经济技术指标」三张核心表集成到一个 Excel 工作簿中，通过 Excel 公式实现表内及表间联动，让用户下载后仍可在本地手动调整关键参数并观察联动结果。

## 二、输出形式

- **页面入口**：导航栏「成果输出」→「规划/产品总表下载」。
- **下载文件**：`规划产品总表.xlsx`。
- **工作簿结构**：共 5 个 sheet。

## 三、Sheet 结构

### Sheet1：规划指标输入

| 字段 | 说明 | 来源 |
|---|---|---|
| 项目名称 | 用户输入 | `projectData.projectName` |
| 城市/区域 | 用户输入 | `projectData.region` |
| 用地面积（㎡） | 用户输入 | `projectData.landArea` |
| 容积率 | 用户输入 | `projectData.far` |
| 限高（m） | 用户输入 | `projectData.heightLimit` |
| 绿地率（%） | 用户输入 | `projectData.greenRate` |
| 配套用房占比（%） | 用户输入 | `projectData.ancillaryRatio` |
| 研发办公占比（%） | 用户输入 | `projectData.rdRatio` |
| 其他规划指标 | 按需补充 | — |

- 该 sheet 为**用户可编辑区域**。

### Sheet2：产品配置选择

| 字段 | 说明 | 来源 |
|---|---|---|
| 选择产品类型 | 轻钢/分栋/分层/产业大厦/配套宿舍/配套楼 | `selectedProducts` |
| 各产品层数 | 如分栋 4F、分层 8F | `productOptions.floors` |
| 各产品面积段 | 如 800/1000/1200 | `productOptions.areas` |
| 其他配置参数 | 如建筑高度、电梯、荷载等 | 产品配置结果 |

- 该 sheet 以**只读形式展示**用户在产品配置页面的选择，用于存档和追溯。

### Sheet3：指标估算

- 该 sheet 完全复刻 `index.html` 生成的《综合指标估算表》。
- **联动规则**：所有数值均通过 Excel 公式引用 Sheet1 的输入值。
  - 例如：`计容总建筑面积 = 用地面积 × 容积率`。
  - 例如：`地上总建筑面积 = 计容总建筑面积`（或按规则调整）。
  - 例如：`建筑密度、绿地面积、道路面积、停车位数量`等均按指标估算算法公式计算。
- 用户修改 Sheet1 后，Sheet3 自动重算。

### Sheet4：产品配置详表

- 该 sheet 完全复刻 `product-config-v2.html` 生成的《产品配置详表》。
- 包含字段：
  - 产品类型、产品形态（独栋/双拼/三拼）、层数、面积段、基底面积、单栋面积、单栋计容、栋数、总建筑面积、总基底面积、总计容面积、电梯配置、荷载、层高（首层/二层/标准层/顶层）、总建筑高度、租金、售价、产品型号等。
- **内部联动**：
  - 修改「首层层高」「二层层高」「标准层层高」「顶层层高」后，自动重算「总建筑高度」。
  - 修改「面积段」「层数」「栋数」后，自动重算「总建筑面积」「总计容面积」。
  - 其他产品配置内部公式按现有规则保留。
- 该 sheet **不与 Sheet1、Sheet3 联动**；用户修改规划指标输入不会反向影响产品配置。

### Sheet5：总体经济技术指标

- 该 sheet 完全复刻 `product-config-v2.html` 生成的《总体经济技术指标表》。
- 包含字段：
  - 规划用地面积、总建筑面积、地上总建筑面积、地下总建筑面积、总计容面积、容积率、建筑密度、绿地率、停车位数量、各产品面积/占比、配套用房面积/占比等。
- **联动规则**：所有数值均通过 Excel 公式引用 Sheet4（产品配置详表）。
  - 例如：`地上总建筑面积 = SUM(各产品总建筑面积)`。
  - 例如：`容积率 = 总计容面积 / 用地面积`。
  - 例如：`建筑密度 = 总基底面积 / 用地面积`。
- 用户修改 Sheet4 后，Sheet5 自动重算。

## 四、联动边界

| 修改位置 | 影响范围 | 不联动范围 |
|---|---|---|
| Sheet1 规划指标输入 | Sheet3 指标估算 | Sheet4、Sheet5 |
| Sheet4 产品配置详表 | Sheet5 总体经济技术指标 | Sheet1、Sheet3 |
| Sheet2 产品配置选择 | 仅展示，不参与计算 | — |

## 五、样式要求

- 表头：深蓝底（#0B2B5C）白字，微软雅黑，加粗。
- 数据区：细边框，微软雅黑，数字右对齐、保留两位小数。
- 合计/关键指标行：浅蓝底或深蓝底白字突出显示。
- 列宽自适应内容。
- 所有数值单元格尽量使用 Excel 公式，支持本地编辑后联动。

## 六、数据来源

- Sheet1/Sheet3：`projectData`（来自 `index.html` 输入并传入 `product-config-v2.html`）。
- Sheet2/Sheet4/Sheet5：`window.configResult`（来自 `product-config-v2.html` 的产品配置结果）。

## 七、异常处理

- 若用户未进行产品配置，Sheet4、Sheet5 为空或提示「请先生成产品配置」。
- 若用户未进行指标估算，Sheet1、Sheet3 仍可基于默认值生成。
