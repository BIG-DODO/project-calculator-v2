# 产品配置算法 - 完整伪代码流程（第七轮修订版）

> **版本**：2026-07-09 第七轮修订版（已自检修补）
> **核心原则**：算法推导全程全部按独栋处理，不考虑双拼转换
> **本轮修复**：
> 1. FL预处理使用 FL 计算结果确定固定栋数（不再每种面积段各固定1栋）
> 2. `mapOptimizedToCounts` 计数改为累加，避免覆盖
> 3. `step5_outer_loop` 中 enrich 后重新计算 `S_type`
> 4. `mapOptimizedToCounts` 新配置按类型所在侧插入，避免索引错位
> 5. `calc_dorm` 最终 base 增加 600-1200 边界钳制
> 6. step0 分层降级到 1 层仍超限高时返回错误
> 7. 增加 `remainingCap <= 0` 检查
> 8. `enrichConfigs` 非轻钢类型增加第二排序键
> 9. `manualDistribute` 按实际类型计算 unitCap
> 10. 补充 `typeSideMap` 维护与传递
> 11. 三种厂房类型处理后清空剩余分层楼层
> 12. `enrichConfigs` 增加 D 参数
> 13. `manualDistribute` 中间面积段 c 增加 [a,b] 钳制
> 14. 3.5层建筑高度统一按4层22.5m计算（已含女儿墙1.2m）
> 15. 产业大厦/配套宿舍等高度公式统一加女儿墙1.2m，宿舍公式修正为 `4.8 + 3.6 * (floors - 1) + 1.2`

---

## 一、全局变量与输入

### 输入参数
- `S` = 规划用地面积
- `F` = 容积率
- `D` = 建筑密度（F<1.5→45%, 1.5≤F<2.0→42%, F≥2.0→40%）
- `R1` = 科研办公用房占比
- `R2` = 配套用房占比
- `Hl` = 限高
- `selectedProducts` = Set，用户已选产品类型
- `productOptions` = 各产品类型的楼层和面积段配置

### 派生变量
- `targetBase = S * D`
- `targetCap = S * F`
- `targetRdArea = targetCap * R1`
- `targetAncillaryArea = targetCap * R2`

## 二、主流程

```
function calculateProductConfig(projectData, selectedProducts, productOptions):

   // ===== 输入验证 =====
   errors = validateInput(projectData, selectedProducts, productOptions)
   if errors.length > 0:
      return { error: errors.join('\n') }

   // ===== 第零步：前置处理 =====
   result0 = step0_preprocess(selectedProducts, productOptions, Hl)
   if result0.error:
      return { error: result0.error }
   selectedProducts = result0.selectedProducts
   productOptions = result0.productOptions
   layerFloors = result0.layerFloors
   splitFloors = result0.splitFloors
   referenceHeight = result0.referenceHeight
   layerDowngraded = result0.layerDowngraded

   // ===== 步骤一：产业大厦 =====
   towerResult = step1_tower(selectedProducts, productOptions, targetRdArea, Hl)
   allConfigs = towerResult.configs
   fixedBase = towerResult.base
   fixedCap = towerResult.cap

   // 若产业大厦存在，用产业大厦高度作为宿舍参考高度
   // 必须在步骤二之前更新，否则产业大厦高度不会影响宿舍计算
   if towerResult.towerHeight > 0:
      referenceHeight = towerResult.towerHeight

   // ===== 步骤二：配套用房 =====
   ancillaryResult = step2_ancillary(selectedProducts, productOptions,
                                     targetAncillaryArea, referenceHeight)
   allConfigs = merge(allConfigs, ancillaryResult.configs)
   fixedBase += ancillaryResult.base
   fixedCap += ancillaryResult.cap

   // ===== 步骤三：厂房预处理 =====
   factoryResult = step3_factory_preprocess(selectedProducts, productOptions,
                                            layerFloors, splitFloors,
                                            targetBase, targetCap,
                                            fixedBase, fixedCap)

   // 若 remainingBase <= 0 或 remainingCap <= 0，说明固定产品已超过上限
   if factoryResult.remainingBase <= 0 或 factoryResult.remainingCap <= 0:
      adjustedResult = adjustFixedConfigs(factoryResult, targetBase, targetCap, fixedBase, fixedCap)
      if !adjustedResult:
         return { error: "固定产品超过建筑密度或容积率上限，请调整输入参数" }
      factoryResult = adjustedResult

   fixedFactoryConfigs = factoryResult.fixedConfigs
   bhConfigs = factoryResult.bhConfigs
   blConfigs = factoryResult.blConfigs
   typeSideMap = factoryResult.typeSideMap
   remainingBase = factoryResult.remainingBase
   remainingCap = factoryResult.remainingCap

   // ===== 步骤四：Bh/Bl核心算法 =====
   coreResult = step4_core_algorithm(bhConfigs, blConfigs,
                                     remainingBase, remainingCap)
   finalCounts = coreResult.counts
   activeBhConfigs = coreResult.bhConfigs
   activeBlConfigs = coreResult.blConfigs
   isSingleType = coreResult.isSingleType
   typeSideMap = coreResult.typeSideMap  // 更新后的映射

   // ===== 步骤五：外层循环优化 =====
   outerResult = step5_outer_loop(finalCounts,
                                  activeBhConfigs, activeBlConfigs,
                                  fixedFactoryConfigs,
                                  remainingBase, remainingCap,
                                  targetBase, targetCap,
                                  fixedBase, fixedCap,
                                  isSingleType, S,
                                  typeSideMap)

   // ===== 输出合并 =====
   output = allConfigs
   for each config in fixedFactoryConfigs:
      if config.count > 0:
         output.push({
            ...config,
            totalCap: config.unitCap * config.count
         })

   for i from 0 to outerResult.counts.length - 1:
      config = outerResult.configs[i]
      count = outerResult.counts[i]
      if count > 0:
         output.push({
            ...config,
            count: count,
            totalCap: config.unitCap * count,
            productType: config.floors + 'F(' + config.base + ')'
         })

   return output
```

## 三、输入验证

```
function validateInput(projectData, selectedProducts, productOptions):

   errors = []

   // 基本参数校验
   if projectData.landArea <= 0: errors.push("用地面积必须大于0")
   if projectData.far <= 0: errors.push("容积率必须大于0")
   if projectData.heightLimit <= 0: errors.push("限高必须大于0")
   if projectData.ancillaryRatio < 0: errors.push("配套占比不能为负")
   if projectData.rdRatio < 0: errors.push("科研办公占比不能为负")
   if projectData.ancillaryRatio + projectData.rdRatio > 1:
      errors.push("配套占比+科研占比不能超过1")

   // 配套用房选择校验
   if projectData.ancillaryRatio > 0:
      if !selectedProducts.has('dorm') 且 !selectedProducts.has('support'):
         errors.push("配套用房未选择：R2>0时必须选择配套楼或配套宿舍")

      totalAncillary = projectData.landArea * projectData.far * projectData.ancillaryRatio
      if totalAncillary > 2400 且 selectedProducts.has('support') 且 !selectedProducts.has('dorm'):
         errors.push("配套用房面积未用尽，请勾选配套宿舍")

   // 厂房类型组合校验
   if selectedProducts.has('layer') 且 selectedProducts.has('light-steel') 且 !selectedProducts.has('split'):
      errors.push("未选择分栋厂房：选择分层厂房和轻钢厂房时必须选择分栋厂房")

   // 层数类型数量校验
   if selectedProducts.has('split') 且 productOptions['split'].floors.length > 2:
      errors.push("分栋厂房层数类型超过2种")
   if selectedProducts.has('layer') 且 productOptions['layer'].floors.length > 2:
      errors.push("分层厂房层数类型超过2种")

   // 面积段选择校验
   for id in selectedProducts:
      if (id == 'light-steel' 或 id == 'split' 或 id == 'layer' 或 id == 'tower' 或 id == 'support'):
         if !productOptions[id].areas 或 productOptions[id].areas.length == 0:
            errors.push("请为「" + getProductName(id) + "」选择面积段")
      if (id == 'split' 或 id == 'layer'):
         if !productOptions[id].floors 或 productOptions[id].floors.length == 0:
            errors.push("请为「" + getProductName(id) + "」选择层数")

   return errors
```

## 四、第零步：前置处理

```
function step0_preprocess(selectedProducts, productOptions, Hl):

   layerFloors = []
   splitFloors = []
   downgradedFloors = []
   layerDowngraded = false
   errors = []

   // 0.1 分层厂房层数降级
   if selectedProducts 包含 'layer':
      for floor in productOptions['layer'].floors:
         currentFloor = floor
         totalH = 7.2 + 5.1 + 4.5 * (currentFloor - 2) + 1.2
         while totalH > Hl and currentFloor > 1:
            currentFloor--
            totalH = 7.2 + 5.1 + 4.5 * (currentFloor - 2) + 1.2

         if totalH <= Hl:
            if currentFloor >= 5:
               layerFloors.push(currentFloor)
               if currentFloor < floor: layerDowngraded = true
            else if currentFloor <= 4:
               downgradedFloors.push(currentFloor)
         else:
            // currentFloor 已降到 1 层仍不满足限高
            errors.push("分层厂房层数=" + floor + "，即使降至1层仍超过限高" + Hl + "m")

      if layerFloors 为空:
         selectedProducts 删除 'layer'

   // 0.2 分栋厂房层数降级
   if selectedProducts 包含 'split':
      for floor in productOptions['split'].floors:
         totalH = computeTotalHeight('split', floor)
         if totalH <= Hl:
            splitFloors.push(floor)
      if splitFloors 为空:
         selectedProducts 删除 'split'

   // 0.3 分层降级为分栋的合并处理
   if downgradedFloors 非空:
      selectedProducts 删除 'layer'
      if selectedProducts 不包含 'split':
         selectedProducts 添加 'split'

      // 原始分层面积段直接合并到分栋（简化规则：不再还原双拼）
      originalLayerAreas = productOptions['layer'].areas
      existingSplitAreas = productOptions['split'].areas 或 []
      mergedAreas = 去重排序(existingSplitAreas + originalLayerAreas)
      productOptions['split'].areas = mergedAreas

      convertedFloors = 去重排序(downgradedFloors)
      existingSplitFloors = productOptions['split'].floors 或 []
      mergedFloors = 去重排序(existingSplitFloors + convertedFloors)
      productOptions['split'].floors = mergedFloors

   // 0.4 更新 productOptions 中的分层参数
   if selectedProducts 包含 'layer' 且 layerFloors 非空:
      productOptions['layer'].floors = layerFloors
      // 面积段保持原始值，不转换

   // 0.5 配套宿舍参考高度上限
   referenceHeight = Hl
   if layerFloors 非空:
      referenceHeight = max(layerFloors.map(f => 7.2 + 5.1 + 4.5 * (f - 2) + 1.2))
   // 注：若产业大厦存在，后续用 towerHeight 回填

   if errors.length > 0:
      return { error: errors.join('\n') }

   return {
      selectedProducts, productOptions,
      layerFloors, splitFloors, referenceHeight, layerDowngraded
   }
```

## 五、步骤一：产业大厦

```
function step1_tower(selectedProducts, productOptions, targetRdArea, Hl):

   configs = []
   base = 0
   cap = 0
   towerHeight = 0

   if selectedProducts 不包含 'tower' 或 targetRdArea <= 0:
      return { configs, base, cap, towerHeight }

   // UI 已限制产业大厦面积段单选，areas[0] 即为用户选择值
   areaRef = productOptions['tower'].areas[0]

   // 第一步：确定初始层数
   floors = max(3, ceil(targetRdArea / areaRef))
   standardHeight = 4.5

   // 第二步：按默认标准层 4.5m 计算建筑高度
   totalH = 6.6 + standardHeight * (floors - 1) + 1.2

   // 第三步：超限后，先尝试降标准层到 4.2m
   if totalH > Hl:
      standardHeight = 4.2
      totalH = 6.6 + standardHeight * (floors - 1) + 1.2

      // 第四步：降标准层后仍超限，恢复 4.5m 并减层数
      if totalH > Hl:
         standardHeight = 4.5
         floors = max(3, floor((Hl - 6.6 - 1.2) / standardHeight) + 1)
         totalH = 6.6 + standardHeight * (floors - 1) + 1.2

         // 第五步：安全兜底
         if totalH > Hl:
            return { error: "限高不足，无法放置3层产业大厦" }

   // 计算面积段
   exactBase = targetRdArea / floors
   finalBase = floor(exactBase / 50) * 50
   actualArea = finalBase * floors

   if |actualArea - targetRdArea| / targetRdArea > 0.002:
      finalBase = floor(exactBase / 20) * 20
      actualArea = finalBase * floors

   paxLift = max(1, ceil(actualArea / 4000) - 2)

   configs.push({
      id: 'tower', type: '产业大厦',
      base: finalBase, unitCap: actualArea, floors, totalHeight: totalH,
      standardHeight,  // 记录最终确定的标准层层高（4.5 或 4.2）
      count: 1, elevator: paxLift + '客2货'
   })

   base = finalBase
   cap = actualArea
   towerHeight = totalH

   return { configs, base, cap, towerHeight }
```

function step2_ancillary(selectedProducts, productOptions,
                         targetAncillaryArea, referenceHeight):

   configs = []
   base = 0
   cap = 0

   if selectedProducts 不包含 'dorm' 且 不包含 'support':
      return { configs, base, cap }

   totalAncillary = targetAncillaryArea

   // 情况1：totalAncillary <= 2400，用户仅选择配套楼
   if totalAncillary <= 2400 且 selectedProducts 包含 'support' 且 不包含 'dorm':
      supportResult = buildSupport(totalAncillary)
      configs.push(supportResult.config)
      base = supportResult.supportBase
      cap = supportResult.supportCap

   // 情况2：totalAncillary <= 2400，用户同时选择配套楼和配套宿舍
   // UI逻辑：两者都选时默认合建为一栋配套楼，全部分配给配套楼
   else if totalAncillary <= 2400 且 selectedProducts 包含 'support' 且 包含 'dorm':
      supportResult = buildSupport(totalAncillary)
      configs.push(supportResult.config)
      base = supportResult.supportBase
      cap = supportResult.supportCap

   // 情况3：totalAncillary <= 2400，用户仅选择配套宿舍
   else if totalAncillary <= 2400 且 selectedProducts 包含 'dorm' 且 不包含 'support':
      dormResult = calc_dorm(totalAncillary, referenceHeight)
      configs.push(dormResult.config)
      base = dormResult.base
      cap = dormResult.cap

   // 情况4：totalAncillary > 2400，用户仅选择配套宿舍
   else if totalAncillary > 2400 且 selectedProducts 包含 'dorm' 且 不包含 'support':
      dormResult = calc_dorm(totalAncillary, referenceHeight)
      configs.push(dormResult.config)
      base = dormResult.base
      cap = dormResult.cap

   // 情况5：totalAncillary > 2400，用户同时选择配套楼和配套宿舍
   else if totalAncillary > 2400 且 selectedProducts 包含 'support' 且 包含 'dorm':
      // 配套楼面积强制限定在 [1200, 2400]
      if totalAncillary < 3600:
         supportCap = 1200
      else:
         // totalAncillary >= 3600 时，使用用户选择的配套楼面积段，但钳制在 [1200, 2400]
         userSupportArea = productOptions['support'].areas[0]
         supportCap = max(1200, min(2400, userSupportArea))
      supportResult = buildSupport(supportCap)
      configs.push(supportResult.config)
      base = supportResult.supportBase
      cap = supportResult.supportCap

      // 剩余给宿舍
      dormCap = totalAncillary - supportResult.supportCap
      if dormCap > 0:
         dormResult = calc_dorm(dormCap, referenceHeight)
         configs.push(dormResult.config)
         base += dormResult.base
         cap += dormResult.cap

   return { configs, base, cap }


function buildSupport(supportCap):

   if supportCap < 1200:
      supportFloors = 2
      totalH = 7.2 + 4.5 + 1.2 = 12.9
      exactBase = supportCap / 2
   else:
      supportFloors = 3
      totalH = 7.2 + 5.1 + 4.5 + 1.2 = 18.3
      exactBase = supportCap / 3

   // 第一次取整：50 的倍数
   supportBase = floor(exactBase / 50) * 50
   finalSupportCap = supportBase * supportFloors

   // 第一次偏差检查
   if |finalSupportCap - supportCap| / supportCap > 0.002:
      // 第二次取整：20 的倍数
      supportBase = floor(exactBase / 20) * 20
      finalSupportCap = supportBase * supportFloors

      // 第二次偏差检查
      if |finalSupportCap - supportCap| / supportCap > 0.002:
         // 第三次取整：10 的倍数
         supportBase = floor(exactBase / 10) * 10
         finalSupportCap = supportBase * supportFloors

   return {
      config: {
         id: 'support', type: '配套楼',
         base: supportBase, unitCap: finalSupportCap,
         floors: supportFloors, totalHeight: totalH, count: 1
      },
      supportCap: finalSupportCap,
      supportBase: supportBase
   }


function calc_dorm(dormCap, referenceHeight):

   estFloors = max(2, floor((referenceHeight - 4.8 - 1.2) / 3.6) + 1)
   singleArea = dormCap / estFloors

   if singleArea < 600:
      base = 600
   else if singleArea > 1200:
      base = 1200
   else:
      base = round(singleArea / 100) * 100

   floors = max(2, ceil(dormCap / base))
   totalH = 4.8 + 3.6 * (floors - 1) + 1.2

   if totalH > referenceHeight:
      totalH = 4.8 + 3.3 * (floors - 1) + 1.2
      if totalH > referenceHeight:
         floors = max(2, floor((referenceHeight - 4.8 - 1.2) / 3.3) + 1)
         totalH = 4.8 + 3.3 * (floors - 1) + 1.2

   exactBase = dormCap / floors
   finalBase = floor(exactBase / 20) * 20

   // 边界钳制：宿舍基底必须在 600-1200 之间
   finalBase = max(600, min(1200, finalBase))

   // 根据钳制后的 base 重新估算层数，尽量接近 dormCap，同时不超过参考高度
   floors = max(2, ceil(dormCap / finalBase))
   totalH = 4.8 + 3.6 * (floors - 1) + 1.2

   if totalH > referenceHeight:
      totalH = 4.8 + 3.3 * (floors - 1) + 1.2
      if totalH > referenceHeight:
         floors = max(2, floor((referenceHeight - 4.8 - 1.2) / 3.3) + 1)
         totalH = 4.8 + 3.3 * (floors - 1) + 1.2

   unitArea = finalBase * floors

   return {
      config: {
         id: 'dorm', type: '配套宿舍',
         base: finalBase, unitCap: unitArea,
         floors, totalHeight: totalH, count: 1
      },
      base: finalBase,
      cap: unitArea
   }
```

## 七、步骤三：厂房预处理

```
function step3_factory_preprocess(selectedProducts, productOptions,
                                  layerFloors, splitFloors,
                                  targetBase, targetCap,
                                  fixedBase, fixedCap):

   fixedConfigs = []
   usedBase = 0
   usedCap = 0
   typeSideMap = {}  // 记录每种类型属于 'bh' 还是 'bl'

   hasLightSteel = selectedProducts 包含 'light-steel'
   hasSplit = selectedProducts 包含 'split'
   hasLayer = selectedProducts 包含 'layer'

   totalLayerTypes = splitFloors.length + layerFloors.length

   // 3.3 FL预处理：当层数类型 > 2 时，固定效率最高（层数最高）的类型
   while totalLayerTypes > 2:
      if layerFloors.length > 1:
         preprocessType = 'layer'
         preprocessFloors = layerFloors
      else if splitFloors.length > 1:
         preprocessType = 'split'
         preprocessFloors = splitFloors
      else:
         break

      maxFloor = max(preprocessFloors)
      maxArea = max(productOptions[preprocessType].areas)
      a = maxArea * maxFloor

      if preprocessType == 'layer':
         FL = targetCap / a / 2
      else:
         FL = targetCap / a

      fixCount = max(1, floor(FL))
      base = maxArea
      floors = maxFloor
      totalH = computeTotalHeight(preprocessType, floors)
      unitCap = computeUnitCap(preprocessType, floors, base)

      fixedConfigs.push({
         id: preprocessType, type: 类型名称,
         base, unitCap, floors, totalHeight: totalH,
         count: fixCount, isFixed: true
      })
      usedBase += base * fixCount
      usedCap += unitCap * fixCount

      preprocessFloors = 过滤掉 maxFloor
      if preprocessType == 'layer':
         layerFloors = preprocessFloors
      else:
         splitFloors = preprocessFloors

      totalLayerTypes = splitFloors.length + layerFloors.length

   // 3.4 三种厂房类型处理
   if hasLightSteel 且 hasSplit 且 hasLayer:
      minLayerFloor = min(layerFloors)
      minLayerArea = min(productOptions['layer'].areas)
      base = minLayerArea
      floors = minLayerFloor
      totalH = computeTotalHeight('layer', floors)
      unitCap = computeUnitCap('layer', floors, base)
      fixedConfigs.push({
         id: 'layer', type: '分层厂房',
         base, unitCap, floors, totalHeight: totalH,
         count: 1, isFixed: true
      })
      usedBase += base
      usedCap += unitCap
      layerFloors = 过滤掉 minLayerFloor

      // 强制分配的这1栋分层即为全部分层，剩余退化为分栋+轻钢
      layerFloors = []

      // 若分栋仍有多种层数，固定最高层数的最大面积段
      if splitFloors.length > 1:
         maxSplitFloor = max(splitFloors)
         maxSplitArea = max(productOptions['split'].areas)
         base = maxSplitArea
         floors = maxSplitFloor
         totalH = computeTotalHeight('split', floors)
         unitCap = computeUnitCap('split', floors, base)
         fixedConfigs.push({
            id: 'split', type: '分栋厂房',
            base, unitCap, floors, totalHeight: totalH,
            count: 1, isFixed: true
         })
         usedBase += base
         usedCap += unitCap
         splitFloors = 过滤掉 maxSplitFloor

   // 3.5 创建 Bh/Bl 配置
   bhConfigs = []
   blConfigs = []

   // allTypes 按层数展开，每个条目代表单一效率层级
   allTypes = []
   if hasLightSteel:
      allTypes.push({ type: 'light-steel', floors: 2.1 })
   if hasSplit 且 splitFloors.length > 0:
      for floor in splitFloors:
         allTypes.push({ type: 'split', floors: floor })
   if hasLayer 且 layerFloors.length > 0:
      for floor in layerFloors:
         allTypes.push({ type: 'layer', floors: floor })

   if allTypes.length == 0:
      // 无厂房
      pass
   else if allTypes.length == 1:
      t = allTypes[0]
      bhConfigs = createConfigs(t.type, t.floors, productOptions)
      for c in bhConfigs: c.side = 'bh'
      typeSideMap[t.type + '_' + t.floors] = 'bh'
   else if allTypes.length == 2:
      t1 = allTypes[0]
      t2 = allTypes[1]
      eff1 = t1.type == 'light-steel' ? 2.1 : t1.floors
      eff2 = t2.type == 'light-steel' ? 2.1 : t2.floors
      if eff1 >= eff2:
         bhConfigs = createConfigs(t1.type, t1.floors, productOptions)
         for c in bhConfigs: c.side = 'bh'
         blConfigs = createConfigs(t2.type, t2.floors, productOptions)
         for c in blConfigs: c.side = 'bl'
         typeSideMap[t1.type + '_' + t1.floors] = 'bh'
         typeSideMap[t2.type + '_' + t2.floors] = 'bl'
      else:
         bhConfigs = createConfigs(t2.type, t2.floors, productOptions)
         for c in bhConfigs: c.side = 'bh'
         blConfigs = createConfigs(t1.type, t1.floors, productOptions)
         for c in blConfigs: c.side = 'bl'
         typeSideMap[t2.type + '_' + t2.floors] = 'bh'
         typeSideMap[t1.type + '_' + t1.floors] = 'bl'
   else:
      // allTypes.length > 2：取效率最高的作为 Bh，效率最低的作为 Bl
      allTypes.sort((a, b) => 效率(b) - 效率(a))
      bhType = allTypes[0]
      blType = allTypes[allTypes.length - 1]
      bhConfigs = createConfigs(bhType.type, bhType.floors, productOptions)
      for c in bhConfigs: c.side = 'bh'
      blConfigs = createConfigs(blType.type, blType.floors, productOptions)
      for c in blConfigs: c.side = 'bl'
      typeSideMap[bhType.type + '_' + bhType.floors] = 'bh'
      typeSideMap[blType.type + '_' + blType.floors] = 'bl'

   remainingBase = targetBase - fixedBase - usedBase
   remainingCap = targetCap - fixedCap - usedCap

   return {
      fixedConfigs, bhConfigs, blConfigs,
      typeSideMap,
      remainingBase, remainingCap
   }


function 效率(t):
   if t.type == 'light-steel':
      return 2.1
   else:
      return t.floors


function createConfigs(type, floor, productOptions):

   configs = []
   areas = productOptions[type].areas 或 []

   for area in areas:
      base = area
      unitCap = computeUnitCap(type, floor, base)
      totalH = computeTotalHeight(type, floor)

      configs.push({
         id: type, type: 类型名称,
         base, unitCap, floors: floor, totalHeight: totalH
      })

   return configs


function computeUnitCap(type, floor, base):
   if type == 'light-steel':
      return base * 2 + 200
   else if type == 'split' 且 floor == 3.5:
      return base * 3.5
   else:
      return base * floor


function computeTotalHeight(type, floor, standardHeight = 4.5):
   if type == 'light-steel':
      return 13.2
   else if floor == 3.5:
      // 3.5层建筑高度按4层计算（22.5m，已含女儿墙1.2m）
      return 22.5
   else if type == 'tower':
      return 6.6 + standardHeight * (floor - 1) + 1.2
   else if type == 'dorm':
      return 4.8 + 3.6 * (floor - 1) + 1.2
   else if floor == 1:
      // split / layer / support 单层情况
      return 7.2 + 1.2
   else:
      // split / layer / support（N>=2 时）
      return 7.2 + 5.1 + 4.5 * (floor - 2) + 1.2


function adjustFixedConfigs(factoryResult, targetBase, targetCap, fixedBase, fixedCap):

   // 从后往前移除 FL 预处理中的固定配置，直到 remainingBase > 0 且 remainingCap > 0
   fixedConfigs = factoryResult.fixedConfigs
   usedBase = sum(fixedConfigs.map(c => c.base * c.count))
   usedCap = sum(fixedConfigs.map(c => c.unitCap * c.count))

   for i from fixedConfigs.length - 1 down to 0:
      if fixedConfigs[i].isFixed:
         usedBase -= fixedConfigs[i].base * fixedConfigs[i].count
         usedCap -= fixedConfigs[i].unitCap * fixedConfigs[i].count
         fixedConfigs.splice(i, 1)
         remainingBase = targetBase - fixedBase - usedBase
         remainingCap = targetCap - fixedCap - usedCap
         if remainingBase > 0 且 remainingCap > 0:
            return {
               fixedConfigs,
               bhConfigs: factoryResult.bhConfigs,
               blConfigs: factoryResult.blConfigs,
               typeSideMap: factoryResult.typeSideMap,
               remainingBase,
               remainingCap
            }

   // 即使移除所有固定配置后仍为负，返回 null
   return null

function step4_core_algorithm(bhConfigs, blConfigs, remainingBase, remainingCap):

   // 处理只有一方配置非空的情况
   if bhConfigs.length == 0 且 blConfigs.length == 0:
      return { counts: [], bhConfigs, blConfigs, isSingleType: true, typeSideMap: {} }

   if bhConfigs.length == 0:
      counts = solveSingleType(blConfigs, remainingCap, remainingBase)
      return { counts, bhConfigs: [], blConfigs, isSingleType: true, typeSideMap: { [blConfigs[0].id]: 'bl' } }

   if blConfigs.length == 0:
      counts = solveSingleType(bhConfigs, remainingCap, remainingBase)
      return { counts, bhConfigs, blConfigs: [], isSingleType: true, typeSideMap: { [bhConfigs[0].id]: 'bh' } }

   effBh = bhConfigs 的平均效率 (unitCap / base)
   effBl = blConfigs 的平均效率
   avgEff = remainingCap / remainingBase
   EPS = 1e-9

   // 4.2 解方程
   if avgEff > effBl + EPS 且 avgEff < effBh - EPS:
      x = (remainingCap - effBl * remainingBase) / (effBh - effBl)
      y = remainingBase - x
      targetCapBh = effBh * x
      targetCapBl = effBl * y
      targetBaseBh = x
      targetBaseBl = y
   else if avgEff >= effBh - EPS:
      targetCapBh = remainingCap
      targetCapBl = 0
      targetBaseBh = remainingBase
      targetBaseBl = 0
   else if avgEff <= effBl + EPS:
      targetCapBh = 0
      targetCapBl = remainingCap
      targetBaseBh = 0
      targetBaseBl = remainingBase
   else:
      x = remainingBase / 2
      y = remainingBase / 2
      targetCapBh = effBh * x
      targetCapBl = effBl * y
      targetBaseBh = x
      targetBaseBl = y

   // 4.3 整数方程求解
   bhResult = solveIntegerEquation(targetCapBh, bhConfigs, targetBaseBh * 1.05)
   blResult = solveIntegerEquation(targetCapBl, blConfigs, targetBaseBl * 1.05)

   finalCounts = [...bhResult.counts, ...blResult.counts]

   // 4.5 取消判断
   bhCount = sum(finalCounts[0..bhConfigs.length-1])
   blCount = sum(finalCounts[bhConfigs.length..])

   typeSideMap = {}
   for c in bhConfigs: typeSideMap[c.id] = 'bh'
   for c in blConfigs: typeSideMap[c.id] = 'bl'

   if bhCount == 0 且 blCount > 0:
      finalCounts = solveSingleType(blConfigs, remainingCap, remainingBase)
      activeBhConfigs = []
      activeBlConfigs = blConfigs
      isSingleType = true
      // 更新映射：Bh 侧类型全部清空
      for c in bhConfigs: delete typeSideMap[c.id]
   else if blCount == 0 且 bhCount > 0:
      finalCounts = solveSingleType(bhConfigs, remainingCap, remainingBase)
      activeBhConfigs = bhConfigs
      activeBlConfigs = []
      isSingleType = true
      // 更新映射：Bl 侧类型全部清空
      for c in blConfigs: delete typeSideMap[c.id]
   else:
      activeBhConfigs = bhConfigs
      activeBlConfigs = blConfigs
      isSingleType = false

   return { counts: finalCounts,
            bhConfigs: activeBhConfigs, blConfigs: activeBlConfigs,
            isSingleType, typeSideMap }


function solveSingleType(configs, targetCap, targetBase):
   result = solveIntegerEquation(targetCap, configs, targetBase * 1.05)
   return result.counts


function solveIntegerEquation(targetCap, configs, targetBaseLimit):

   hasBaseLimit = targetBaseLimit != null 且 targetBaseLimit > 0
   realTargetBase = hasBaseLimit ? targetBaseLimit / 1.05 : 0
   k = configs.length

   if k == 0:
      return { counts: [], totalCap: 0, totalBase: 0 }

   if k == 1:
      c = configs[0]
      n = max(1, round(targetCap / c.unitCap))
      bestN = n
      bestScore = infinity
      searchRange = max(50, floor(n * 0.5))
      startN = max(1, n - searchRange)
      endN = n + searchRange
      for testN from startN to endN:
         testCap = c.unitCap * testN
         testBase = c.base * testN
         if hasBaseLimit 且 testBase > targetBaseLimit * 1.05: continue
         score = |testCap - targetCap| + |testBase - realTargetBase| * 2
         if score < bestScore:
            bestScore = score
            bestN = testN
      return { counts: [bestN],
               totalCap: c.unitCap * bestN,
               totalBase: c.base * bestN }

   // 多配置 DFS
   best = null
   bestScore = infinity

   getSearchRange(targetVal, unitVal):
      estN = max(0, round(targetVal / unitVal))
      if estN <= 3: return 4
      if estN <= 10: return 8
      if estN <= 30: return 15
      return 30

   dfs(idx, current, currentCap, currentBase):
      if idx == k - 1:
         c = configs[idx]
         n = max(0, round((targetCap - currentCap) / c.unitCap))
         range = getSearchRange(targetCap - currentCap, c.unitCap)
         for dn from -range to range:
            nn = max(0, n + dn)
            tc = currentCap + c.unitCap * nn
            tb = currentBase + c.base * nn
            if hasBaseLimit 且 tb > targetBaseLimit * 1.05: continue
            if hasBaseLimit 且 currentBase > targetBaseLimit * 1.5: return
            if currentCap > targetCap * 1.5: return

            capDiff = |tc - targetCap|
            baseDiff = hasBaseLimit ? |tb - realTargetBase| : 0
            allCounts = [...current, nn]
            nonZeroCount = 非零个数
            penalty = (k >= 2 且 nonZeroCount < 2) ? 50000 : 0
            score = capDiff + baseDiff * 2 + penalty

            if score < bestScore:
               bestScore = score
               best = [...current, nn]
         return

      c = configs[idx]
      n = max(0, round((targetCap - currentCap) / c.unitCap))
      range = getSearchRange(targetCap - currentCap, c.unitCap)
      for dn from -range to range:
         nn = max(0, n + dn)
         nextBase = currentBase + c.base * nn
         nextCap = currentCap + c.unitCap * nn
         if hasBaseLimit 且 nextBase > targetBaseLimit * 1.5: continue
         if nextCap > targetCap * 1.5: continue
         dfs(idx + 1, [...current, nn], nextCap, nextBase)

   dfs(0, [], 0, 0)

   if !best:
      // fallback：用最小面积段
      sortedConfigs = configs 按 base 排序
      minBase = sortedConfigs[0].base
      maxN = hasBaseLimit ? floor(targetBaseLimit * 1.05 / minBase) : 100
      if maxN <= 0:
         // 即使密度可能超限，也保证至少 2 栋最小面积段
         result = configs.map(() => 0)
         minIdx = configs 中 sortedConfigs[0] 的索引
         result[minIdx] = 2
         return {
            counts: result,
            totalCap: sortedConfigs[0].unitCap * 2,
            totalBase: sortedConfigs[0].base * 2
         }

      bestN = max(1, min(maxN, round(targetCap / sortedConfigs[0].unitCap)))
      bestScore = infinity
      for n from 1 to maxN:
         tc = sortedConfigs[0].unitCap * n
         tb = sortedConfigs[0].base * n
         score = |tc - targetCap| + |tb - realTargetBase| * 2
         if score < bestScore:
            bestScore = score
            bestN = n

      result = configs.map(() => 0)
      minIdx = configs 中 sortedConfigs[0] 的索引
      result[minIdx] = bestN
      return { counts: result,
               totalCap: sortedConfigs[0].unitCap * bestN,
               totalBase: sortedConfigs[0].base * bestN }

   totalCap = sum(c.unitCap * best[i])
   totalBase = sum(c.base * best[i])
   return { counts: best, totalCap, totalBase }

## 九、步骤五：外层循环

```
function step5_outer_loop(finalCounts, bhConfigs, blConfigs,
                          fixedFactoryConfigs,
                          remainingBase, remainingCap,
                          targetBase, targetCap,
                          fixedBase, fixedCap,
                          isSingleType, landArea,
                          typeSideMap):

   counts = [...finalCounts]
   currentBhConfigs = [...bhConfigs]
   currentBlConfigs = [...blConfigs]
   currentTypeSideMap = { ...typeSideMap }
   oscillationHistory = []

   // 重新计算实际可用 remaining（考虑固定配置）
   usedBase = sum(fixedFactoryConfigs.map(c => c.base * c.count))
   usedCap = sum(fixedFactoryConfigs.map(c => c.unitCap * c.count))
   currentRemainingBase = remainingBase
   currentRemainingCap = remainingCap

   CAP_LO = targetCap * 0.995
   CAP_HI = targetCap * 1.005

   for round from 0 to 9:

      totals = calcTotals(counts, currentBhConfigs, currentBlConfigs, fixedFactoryConfigs)
      totals.base += fixedBase
      totals.cap += fixedCap

      // 振荡检测
      densityDiff = totals.base - targetBase
      if densityDiff > targetBase * 0.05: oscillationHistory.push(1)
      else if densityDiff < -targetBase * 0.05: oscillationHistory.push(-1)
      else: oscillationHistory.push(0)
      if oscillationHistory.length > 5: oscillationHistory.shift()

      // 分布检查
      allConfigs = [...currentBhConfigs, ...currentBlConfigs]
      distCheck = checkDistribution(counts, allConfigs)

      if !distCheck.needsOptimization 且 !detectOscillation(oscillationHistory):
         if totals.cap >= CAP_LO 且 totals.cap <= CAP_HI:
            break

      // 需要优化
      if distCheck.needsOptimization 或 detectOscillation(oscillationHistory):
         type = distCheck.type
         a = distCheck.a
         b = distCheck.b
         floor = distCheck.floor

         typeIndices = allConfigs 中 id == type 的索引
         S_type = sum(allConfigs[i].base * counts[i])  // i in typeIndices

         if isSingleType:
            targetConfigs = currentBhConfigs.length > 0 ? currentBhConfigs : currentBlConfigs

            if targetConfigs.length == 0: continue

            totalCount = sum(counts)
            enrichedConfigs = enrichConfigs(targetConfigs, landArea, currentRemainingBase, totalCount)
            newCounts = solveIntegerEquation(currentRemainingCap, enrichedConfigs, currentRemainingBase * 1.05)

            if targetConfigs == currentBhConfigs:
               currentBhConfigs = enrichedConfigs
            else:
               currentBlConfigs = enrichedConfigs
            counts = newCounts.counts

            // enrich 后重新计算 S_type
            allConfigs = [...currentBhConfigs, ...currentBlConfigs]
            typeIndices = allConfigs 中 id == type 的索引
            S_type = sum(allConfigs[i].base * counts[i])

            totals = calcTotals(counts, currentBhConfigs, currentBlConfigs, fixedFactoryConfigs)
            totals.base += fixedBase
            totals.cap += fixedCap

            distCheck2 = checkDistribution(counts, [...currentBhConfigs, ...currentBlConfigs])
            if distCheck2.needsOptimization 或 detectOscillation(oscillationHistory):
               optimized = manualDistribute(type, floor, S_type, a, b)
               counts = mapOptimizedToCounts(optimized, counts, currentBhConfigs, currentBlConfigs, type, currentTypeSideMap)
         else:
            optimized = manualDistribute(type, floor, S_type, a, b)
            counts = mapOptimizedToCounts(optimized, counts, currentBhConfigs, currentBlConfigs, type, currentTypeSideMap)

      // 精细调节
      fineTuned = fineTune(counts, [...currentBhConfigs, ...currentBlConfigs],
                           targetCap, targetBase, fixedBase, fixedCap)
      counts = fineTuned.counts
      totals = fineTuned.totals

      if totals.cap >= CAP_LO 且 totals.cap <= CAP_HI 且 |totals.base - targetBase| <= targetBase * 0.05:
         break

   return { counts, configs: [...currentBhConfigs, ...currentBlConfigs] }


function calcTotals(counts, bhConfigs, blConfigs, fixedFactoryConfigs):
   base = 0; cap = 0
   for c in fixedFactoryConfigs:
      base += c.base * c.count
      cap += c.unitCap * c.count
   allConfigs = [...bhConfigs, ...blConfigs]
   for i from 0 to counts.length - 1:
      if allConfigs[i] 且 counts[i] > 0:
         base += allConfigs[i].base * counts[i]
         cap += allConfigs[i].unitCap * counts[i]
   return { base, cap }


function checkDistribution(counts, configs):

   typeGroups = {
      'light-steel': { indices: [], totalCount: 0 },
      'split': { indices: [], totalCount: 0 },
      'layer': { indices: [], totalCount: 0 }
   }

   for i from 0 to configs.length - 1:
      if counts[i] > 0:
         typeGroups[configs[i].id].indices.push(i)
         typeGroups[configs[i].id].totalCount += counts[i]

   // 检查是否需要优化
   for type in ['light-steel', 'split', 'layer']:
      group = typeGroups[type]
      if group.totalCount < 5: continue
      if group.indices.length == 0: continue
      for idx in group.indices:
         areaCount = counts[idx]
         rest = group.totalCount - areaCount
         if areaCount > rest * 2:
            return {
               needsOptimization: true,
               type,
               a: min(group.indices.map(i => configs[i].base)),
               b: max(group.indices.map(i => configs[i].base)),
               floor: configs[group.indices[0]].floors
            }

   // 不需要优化时，返回主要类型信息
   dominantType = null
   dominantCount = 0
   for type in ['light-steel', 'split', 'layer']:
      if typeGroups[type].totalCount > dominantCount:
         dominantCount = typeGroups[type].totalCount
         dominantType = type

   if dominantType 且 typeGroups[dominantType].indices.length > 0:
      dIndices = typeGroups[dominantType].indices
      return {
         needsOptimization: false,
         type: dominantType,
         a: min(dIndices.map(i => configs[i].base)),
         b: max(dIndices.map(i => configs[i].base)),
         floor: configs[dIndices[0]].floors
      }

   return { needsOptimization: false, type: null, a: null, b: null, floor: null }


function manualDistribute(type, floor, S, a, b):

   eff = 实际效率(type, floor, (a + b) / 2)

   if S < b:
      base = ceil(S / 10) * 10
      return [{ base, count: 1, unitCap: computeUnitCap(type, floor, base) }]

   refCount = ceil(S * 2 / (a + b))

   if refCount <= 3:
      unifiedBase = ceil(S / refCount / 10) * 10
      return [{ base: unifiedBase, count: refCount, unitCap: computeUnitCap(type, floor, unifiedBase) }]

   else if refCount <= 6:
      aa = max(1, ceil(refCount * 2 / 3) - 2)
      bb = 1
      cc = refCount - aa - bb
      if cc <= 0:
         unifiedBase = ceil(S / refCount / 10) * 10
         return [{ base: unifiedBase, count: refCount, unitCap: computeUnitCap(type, floor, unifiedBase) }]

      S1 = a * aa + b * bb
      remaining = S - S1
      c = ceil(remaining / cc / 10) * 10
      c = max(a, min(b, c))  // 中间面积段必须在 [a, b] 之间
      return [
         { base: a, count: aa, unitCap: computeUnitCap(type, floor, a) },
         { base: c, count: cc, unitCap: computeUnitCap(type, floor, c) },
         { base: b, count: bb, unitCap: computeUnitCap(type, floor, b) }
      ]

   else:
      aa = max(1, ceil(refCount * 2 / 3) - 2)
      bb = 2
      S1 = a * aa + b * bb
      remaining = S - S1
      if remaining <= 0:
         return [
            { base: a, count: aa, unitCap: computeUnitCap(type, floor, a) },
            { base: b, count: bb, unitCap: computeUnitCap(type, floor, b) }
         ]
      cRef = (a + b) / 2
      cc = max(1, ceil(remaining / cRef))
      c = ceil(remaining / cc / 10) * 10
      c = max(a, min(b, c))  // 中间面积段必须在 [a, b] 之间
      return [
         { base: a, count: aa, unitCap: computeUnitCap(type, floor, a) },
         { base: c, count: cc, unitCap: computeUnitCap(type, floor, c) },
         { base: b, count: bb, unitCap: computeUnitCap(type, floor, b) }
      ]


function 实际效率(type, floor, base):
   return computeUnitCap(type, floor, base) / base


function mapOptimizedToCounts(optimized, counts, bhConfigs, blConfigs, type, typeSideMap):

   allConfigs = [...bhConfigs, ...blConfigs]

   // 找到 type 对应的索引范围
   typeIndices = []
   for i from 0 to allConfigs.length - 1:
      if allConfigs[i].id == type: typeIndices.push(i)

   // 清空原有 counts（累加前先归零）
   for i in typeIndices: counts[i] = 0

   // 映射 optimized 结果
   for opt in optimized:
      bestIdx = -1
      bestDiff = infinity
      for i in typeIndices:
         diff = |allConfigs[i].base - opt.base|
         if diff < bestDiff:
            bestDiff = diff
            bestIdx = i

      if bestIdx >= 0:
         // 累加，避免多个 opt 映射到同一配置时覆盖
         counts[bestIdx] += opt.count
      else:
         // 若 opt.base 不存在于 configs，添加新配置
         newConfig = {
            id: type, type: 类型名称,
            base: opt.base, unitCap: opt.unitCap,
            floors: allConfigs[typeIndices[0]].floors,
            totalHeight: allConfigs[typeIndices[0]].totalHeight
         }

         // 按 typeSideMap 决定插入 Bh 还是 Bl
         side = typeSideMap[type]
         if side == 'bh':
            insertIdx = bhConfigs.length
            bhConfigs.push(newConfig)
            counts.splice(insertIdx, 0, opt.count)
         else if side == 'bl':
            blConfigs.push(newConfig)
            counts.push(opt.count)
         else:
            // 兜底：若 type 不在映射中，默认放入 bh
            insertIdx = bhConfigs.length
            bhConfigs.push(newConfig)
            counts.splice(insertIdx, 0, opt.count)

   return counts


function enrichConfigs(configs, landArea, targetBaseForType, estimatedTotalCount):

   if configs.length == 0: return configs

   if landArea <= 20000: maxAdd = 1
   else if landArea <= 50000: maxAdd = 2
   else: maxAdd = 3

   type = configs[0].id
   floor = configs[0].floors
   candidates = new Set()

   // 递减法（上限放宽到当前最大面积段 + 500，不再硬限制 1200）
   maxBase = max(configs.map(c => c.base))
   enrichUpper = maxBase + 500
   for config in configs:
      x = config.base
      if x - 50 >= 300: candidates.add(x - 50)
      if x + 50 <= enrichUpper: candidates.add(x + 50)
      if x - 100 >= 300: candidates.add(x - 100)

   // 内插法
   sortedAreas = configs.map(c => c.base).sort((a, b) => a - b)
   for i from 0 to sortedAreas.length - 2:
      x1 = sortedAreas[i]
      x2 = sortedAreas[i + 1]
      mid = round((x1 + x2) / 2 / 50) * 50
      if mid != x1 且 mid != x2: candidates.add(mid)

   // 过滤和排序
   existingAreas = new Set(configs.map(c => c.base))

   // 目标基底：传入该类型的实际目标基底（如 currentRemainingBase / targetBaseBh / targetBaseBl）
   // estimatedTotalCount 为该类型的预估总栋数，由调用方根据当前求解结果计算
   estCount = max(1, estimatedTotalCount)
   targetBaseApprox = targetBaseForType / estCount

   filteredCandidates = [...candidates]
      .filter(area => !existingAreas.has(area))
      .map(area => {
         eff = computeUnitCap(type, floor, area) / area
         targetEff = configs[0].unitCap / configs[0].base
         effDiff = |eff - targetEff|
         baseDiff = |area - targetBaseApprox|
         return { area, effDiff, baseDiff }
      })
      .sort((a, b) => {
         // 轻钢优先按效率差排序；非轻钢效率相同，按 base 差排序
         if a.effDiff != b.effDiff: return a.effDiff - b.effDiff
         return a.baseDiff - b.baseDiff
      })
      .slice(0, maxAdd)
      .map(c => c.area)

   // 创建新配置
   newConfigs = []
   for area in filteredCandidates:
      base = area
      unitCap = computeUnitCap(type, floor, base)
      totalH = computeTotalHeight(type, floor)

      newConfigs.push({
         id: type, type: configs[0].type,
         base, unitCap, floors: floor, totalHeight: totalH,
         isEnriched: true
      })

   enrichedConfigs = [...configs, ...newConfigs]
   enrichedConfigs.sort((a, b) => a.base - b.base)
   return enrichedConfigs


function fineTune(counts, configs, targetCap, targetBase, fixedBase, fixedCap):

   MAX_FINE_TUNE = 5
   previousCounts = null

   for round from 0 to MAX_FINE_TUNE - 1:
      totals = calcTotals(counts, configs, [], [])
      totals.base += fixedBase
      totals.cap += fixedCap

      farDiff = totals.cap - targetCap
      densityDiff = totals.base - targetBase

      if |farDiff| <= targetCap * 0.0001 且 |densityDiff| <= targetBase * 0.05:
         return { counts, totals }

      if |densityDiff| > targetBase * 0.05:
         if previousCounts: counts = previousCounts
         return { counts, totals }

      targetEff = targetCap / targetBase
      bestIdx = -1
      bestScore = infinity
      for i from 0 to configs.length - 1:
         if counts[i] <= 0: continue
         eff = configs[i].unitCap / configs[i].base
         score = |eff - targetEff|
         if score < bestScore:
            bestScore = score
            bestIdx = i

      if bestIdx < 0: break

      previousCounts = [...counts]

      if farDiff > 0:
         if counts[bestIdx] > 1: counts[bestIdx]--
      else:
         counts[bestIdx]++

      newTotals = calcTotals(counts, configs, [], [])
      newTotals.base += fixedBase
      if newTotals.base > targetBase * 1.05:
         counts = previousCounts
         return { counts, totals }

   finalTotals = calcTotals(counts, configs, [], [])
   finalTotals.base += fixedBase
   finalTotals.cap += fixedCap
   return { counts, totals: finalTotals }


function detectOscillation(history):
   if history.length < 3: return false
   last3 = history[-3:]
   return (last3[0] > 0 且 last3[1] < 0 且 last3[2] > 0) 或
          (last3[0] < 0 且 last3[1] > 0 且 last3[2] < 0)
```
