# 静态投资分析设计文档

## 一、设计目标

根据产品配置结果、投资估算数据和用户输入的租售价格，自动计算项目静态投资回报指标，形成：
- **页面端展示**：销售测算表、租赁测算表、租售面积分配表、综合汇总指标表。
- **下载表格**：Excel 多 sheet 表格，带公式、可编辑。

---

## 二、计算流程

```
产品配置结果 + 投资估算数据 + 用户输入租售价格
        ↓
销售面积分配 / 租赁面积分配（按优先级）
        ↓
加权平均售价 / 加权平均租金（保留 6 位中间精度）
        ↓
销售测算 / 租赁测算
        ↓
综合汇总指标
```

## 四、核心计算规则

### 面积分配

```
可销售计容面积 = 计容总建筑面积 × (1 - 配套占比 - 研发办公占比)
销售面积 = 可销售计容面积 × 可分割销售比例
可租面积 = 地上总建筑面积 - 销售面积 - 配套楼面积
自用面积 = 配套楼面积
```

### 销售面积分配优先级

```
轻钢 → 分栋 → 分层 → 产业大厦
```

### 租赁面积分配规则

1. **配套宿舍强制全部出租**：只要产品配置中存在配套宿舍，其全部面积即计入出租面积，不参与出租优先级竞争。即使在全售模式下，配套宿舍也作为出租物业处理。
2. **剩余可租面积分配优先级**：
   ```
   产业大厦 → 分层 → 分栋 → 轻钢
   ```
3. **分配流程**：
   ```
   dormArea = 配套宿舍建筑面积（强制出租）
   remainingRentable = 可租面积 - dormArea
   按优先级将 remainingRentable 分配给产业大厦、分层、分栋、轻钢
   最终出租面积 = dormArea + 各厂房类产品出租面积之和
   ```

### 加权平均价格

```
加权平均售价 = Σ（各产品售价 × 各产品销售面积） ÷ 总销售面积
加权平均租金 = Σ（各产品租金 × 各产品可租面积） ÷ 总可租面积
```

- 租售面积分配表中的「加权平均售价/租金」使用 Excel 公式计算，**保留 6 位小数**，避免后续销售收入累计误差；
- 销售测算表中的「加权平均售价」、租赁测算表中的「加权平均租金」必须引用租售面积分配表的合计行，禁止直给；
- 页面展示时四舍五入保留两位小数。

### 财务费用分摊规则

```
总财务费用 = 投资估算利息支出 + 银行费用支出
销售分摊财务费用 = 总财务费用 × 销售面积 ÷ (销售面积 + 可租面积)
租赁分摊财务费用 = 总财务费用 × 可租面积 ÷ (销售面积 + 可租面积)
```

- 配套楼自用面积不参与财务费用分摊；
- 销售测算表中「减：财务费用」引用销售分摊额；
- 租赁测算表中「租赁总投」包含租赁分摊财务费用。

### 6.3 销售测算

| 项目 | 单位 | 公式 |
|------|------|------|
| 可售面积 | ㎡ | 租售面积分配表合计 |
| 加权平均售价 | 万元/㎡ | 租售面积分配表合计（引用） |
| 不含税售价 | 万元/㎡ | 加权平均售价 ÷ 1.09 |

| 减：建安成本 | 万元 | 可售面积 × 建安成本单方（前期+建安工程）÷ 10000 |
| 减：税金及附加 | 万元 | 不含税收入 × 0.6% |
| 减：土地增值税 | 万元 | 引用土地增值税测算表最终税额 |
| 减：营销费用 | 万元 | 销售收入 × 营销费用率 |
| 减：管理费用 | 万元 | 销售收入 × 管理费用率 |
| 减：财务费用 | 万元 | 投资估算财务费用 × 销售面积 ÷ (销售面积 + 可租面积) |
| 利润总额 | 万元 | 销售收入 - 各项扣除 |
| 所得税 | 万元 | MAX(利润总额 × 25%, 0) |
| 净利润 | 万元 | 利润总额 - 所得税 |
| 销售净利率 | % | 净利润 ÷ 销售收入 |

## 九、综合汇总指标表

| 指标 | 单位 | 公式 |
|------|------|------|
| 总投资 | 万元 | 投资估算的发展成本合计 |

## 十、Excel 下载要求

- **格式**：Excel，多 sheet。
- **Sheet 1：销售测算**：规划指标、建造成本、销售测算结果。
- **Sheet 2：租赁测算**：规划指标、建造成本、租赁测算结果。

- 所有数值单元格使用 Excel 公式引用基础数据，支持用户修改后自动重算。
- 销售测算表中的「减：土地增值税」直接引用土地增值税测算表的最终税额。
- 所有求和公式统一包 `ROUND(SUM(...), 2)`，中间计算结果也包 `ROUND(..., 2)`。
- 所有数字单元格显式设置 `t='n'`，字符串单元格设置 `t='s'`，避免被错误序列化为 `t='str'` 导致 `#VALUE!`。

## 十三、已确认事项

1. 年租金收入统一采用 **360 天**口径，即月租金×12 = 日租金×360。
2. 租售面积分配表中的加权平均售价/租金保留 **6 位小数**，展示时保留两位。
3. 综合汇总中「资金缺口」已改为「资金盈余/缺口」，公式为 `总投资 × 融资占比 - 销售收入`；正值显示资金缺口，负值显示资金盈余。
4. 租赁总投包含可租面积对应建造成本及租赁分摊的财务费用。
5. 静态投资回收期 = 总投资 ÷ 租赁年净收入。
6. 财务费用按销售面积与可租面积比例分摊，配套楼自用面积不分摊。
7. 销售成本拆分采用「方案B」口径（2026-07-20 确认）：土地成本 = 可售面积 × 土地成本单方（土地配套费合计 ÷ 地上总建筑面积）；建安成本 = 可售面积 × 建安成本单方（前期费用+建安工程成本 ÷ 总建筑面积）。两行互补不重叠、无遗漏，修正原「土地含契税 + 综合建造成本」口径下出让金与契税被重复计扣的问题；综合建造成本（含土地）仅保留用于租赁总投口径。
7. 销售测算表中的销售收入、加权平均售价、可售面积均引用租售面积分配表合计行。
8. 所有数字单元格显式标记类型，避免启用编辑后出现 `#VALUE!`。
