# 动态投资分析设计文档

## 一、设计目标

在静态投资分析基础上，引入时间维度，计算项目多年现金流、NPV、IRR、自有资金 IRR、动态投资回收期，并提供敏感性分析。

---

## 二、与其他模块的联动关系

```
产品配置结果 / 综合指标表
        ↓
投资估算表
        ↓
静态投资分析表
        ↓
动态投资分析表
```

- 若用户未完成投资估算，提示「请先进行投资估算」；
- 若用户已完成投资估算但未完成静态投资分析，提示「请先进行静态投资分析」。

## 七、动态模型核心计算逻辑

### 7.1 基本假设

1. **默认不考虑预售**：预售开关默认关闭，但保留开启能力；
2. **去化口径**：销售/租赁均按年去化速度（万㎡/年）逐年释放，直到卖完或满租；
3. **满租定义**：累计租赁去化面积 ≥ 可租面积 × 稳定出租率（默认 90%）即停止租赁去化，剩余面积作为结构性空置，不再产生收入；
4. **预租与预售回款时点不同**：
   - 预售去化当年即确认回款（面积 × 当年售价），并计入土增税/所得税清算基数；
   - 预租（建设期租赁去化）只是锁定面积，因竣备后才能签租赁合同，租金收入从运营期第 1 年才开始计提（含预租部分）；
5. **清算时点**：税金及附加随销售按年计提；土地增值税和企业所得税在**销售去化完成当年**一次性清算（原「建设期末统一清算」修正为此口径，开预售时清算基数含建设期预售部分）；
6. **全部费用化**：运营期利息按剩余本金 × 融资利率逐年费用化，不留资本化余额；
7. **不考虑建设期租金收益**：工业项目竣备前通常不能出租（见第 4 条）。

### 7.2 去化模型

**不开预售**：建设期 0 去化；运营期第 1 年起，每年去化 `min(去化速度, 剩余可去化量)`，销售直到卖完、租赁直到满租。

**开预售**：建设期第 1、2 年按用户输入的去化比例 × 总可售/可租量去化（默认 0%、30%）；运营期起对**剩余量**按去化速度继续去化。

**示例**：销售物业 10 万方，租赁物业 10 万方，去化速度 3.0 万㎡/年，稳定出租率 90%。

- 不开预售：
  - 运营期第 1 年：销售去化 3 万方（余 7），租赁去化 3 万方（余 7）；
  - 第 2 年：销售 3（余 4），租赁 3（余 4）；
  - 第 3 年：销售 3（余 1），租赁 3（余 1，累计 9 万方 = 90%，满租，租赁停止）；
  - 第 4 年：销售 1，去化完成。
- 开预售（建设期第 1 年 20%、第 2 年 30%）：
  - 建设期第 1 年：销售去化 2 万方（余 8），租赁预租 2 万方（余 8）；
  - 建设期第 2 年：销售 3（余 5），租赁预租 3（余 5）；
  - 运营期第 1 年：销售 3（余 2），租赁 3（余 2）；
  - 运营期第 2 年：销售 2（去化完成），租赁 1（累计 9 万方 = 90%，满租）。

### 7.3 三阶段现金流模型

**阶段一：建设期**

- 自有资金：总投资 × (1 − 融资占比)，按建设期年数均匀投入；
- 银行贷款：按工程进度均匀提款；**预售回款优先替代贷款提款**（贷款提款 = 投资支出 − 自有资金 − 预售回款，提款上限 = 总投资 × 融资占比）；
- 当年利息 = 已提本金 × 融资利率（建设期利息已含在投资估算财务费用中，建设期不再重复计息；运营期利息见阶段二/三）；
- 年份序号：建设期第 1 年 = 第 0 年，依次顺延。

**阶段二：销售 + 运营混合期（销售完成前）**

- 可用资金 = 销售回款 + 租金收入 − 销售税费 − 运营税费 − 运营成本；
- 利息（必须支付）= 剩余本金 × 融资利率；
- 还本 = min(可用资金 − 利息, 剩余本金)；
- 股东可分配 = 可用资金 − 利息 − 还本；
- 若可用资金 − 利息 < 0 → 股东需补足（股东可分配为负），页面端对该年份做预警标识，不阻断计算。

**阶段三：纯运营期（销售完成后）**

- 可用资金 = 租金收入 − 运营税费 − 运营成本；
- 利息 = 剩余本金 × 融资利率；
- 还本 = min(可用资金 − 利息, 剩余本金)；
- 股东可分配 = 可用资金 − 利息 − 还本。

**自有资金现金流**：

- 建设期：自有资金现金流 = −当年自有资金出资额；
- 运营期：自有资金现金流 = 股东可分配。

### 7.4 融资与利息

- 融资方案与投资估算联动（融资利率、融资占比）；
- 贷款本金上限 = 总投资 × 融资占比；开预售时实际提款 = 总投资 × 融资占比 − 建设期预售回款（下限 0）；
- 运营期全部费用化：当年利息 = 剩余本金 × 融资利率，随还本逐年递减。

### 7.5 税金清算

- 税金及附加：随各年销售回款按年计提（口径同静态分析：销售回款 / 1.09 × 0.006）；
- 土地增值税：按静态分析四级超率累进口径，以全部销售收入（含建设期预售）为基数，在销售去化完成当年一次性清算；
- 企业所得税：以销售利润总额为基数 × 25%，与土增税同年一次性清算；
- 租赁部分不计所得税，与静态分析口径一致。

### 7.6 核心输出指标

| 指标 | 公式 / 口径 |
|------|------|
| 项目 NPV | Σ（第 t 年项目净现金流 / (1 + 折现率)^t），项目净现金流 = 可用资金（建设期为 −总投资支出） |
| 项目 IRR | 使项目 NPV = 0 的折现率 |
| 自有资金 IRR | 使自有资金 NPV = 0 的折现率（现金流见 7.3） |
| 动态投资回收期 | 项目累计折现现金流由负转正的年份 |

> NPV / IRR 在 JS 端用数值方法（二分 / 牛顿迭代）求解；Excel 端使用内置 `NPV()` / `IRR()` 函数并写入 JS 预计算缓存值。

## 十、已确认 / 待细化事项

### 已确认事项

1. 动态模型默认不考虑预售，但预留预售开关；建设期第 1/2 年去化比例默认 0% / 30%，作用于销售与租赁总量。
2. 销售/租赁去化速度提供选项：1.0 / 1.2 / 1.5 / 1.8 / 2.0 / 2.5 / 3.0 万㎡/年；满租 = 累计租赁去化 ≥ 可租面积 × 稳定出租率（90%）。
3. 取消「租赁爬坡年数」输入，爬坡期由去化速度推导。
4. 预售当年确认回款并计入清算基数；预租只锁定面积，租金收入从运营期第 1 年计提（竣备后才能签租赁合同）。
5. 税金及附加随销售按年计提；土增税、所得税在**销售去化完成当年**一次性清算（替代原「建设期末统一清算」）。
6. 采用三阶段现金流模型：建设期（自有资金均匀投入、贷款按进度提款、预售回款优先替代贷款提款）→ 销售+运营混合期 → 纯运营期；利息 = 剩余本金 × 融资利率，还本 = min(可用资金 − 利息, 剩余本金)，股东可分配 = 可用资金 − 利息 − 还本，为负即股东补足并做预警标识。
7. 全部费用化，运营期不留融资余额。
8. 自有资金现金流：建设期 = −当年自有出资额；运营期 = 股东可分配；据此计算自有资金 IRR。
9. 敏感性分析 6 个变量：售价、租金、建安成本、土地价格、出租率、去化周期，各 ±10%/±20%；UI 只显示售价/租金/建安 3 组，Excel 展示全部 6 组。
10. Excel 按年展示，页面端只展示关键年份（0、1、2、5、10、20 年）。
11. Excel 采用「参数区可编辑 + 公式链 + 内置 NPV()/IRR() + JS 缓存值」结构，手动改参数后全表自动重算。
12. 增加自有资金 IRR 指标。
13. 资金缺口公式沿用静态分析：`总投 - (1 - 融资占比) × 总投 - 销售收入`。

### 待细化事项

1. 建设期支出在各年的具体分配比例（默认均匀投入，后续可根据付款节点细化）。
2. 敏感性分析 tornado 图是否按影响程度自动排序（当前按影响程度排序实现）。
3. 开预售时「贷款提款上限 − 预售回款」若出现负值（预售超贷款额度）的处理（当前按下限 0 处理，多余预售回款视同留存项目）。
