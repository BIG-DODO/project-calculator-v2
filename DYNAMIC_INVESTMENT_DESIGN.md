# 动态投资分析设计文档（方向版）

## 一、设计目标

在静态投资分析基础上，引入时间维度，计算项目多年现金流、NPV、IRR、自有资金 IRR、动态投资回收期，并提供敏感性分析。

---

## 二、与其他模块的联动关系

```
产品配置结果 / 综合指标表
        ↓
投资估算表
        ↓
静态投资分析表
        ↓
动态投资分析表
```

## 三、动态分析用户输入项

| 输入项 | 类型 | 默认值 | 是否必填 | 说明 |
|--------|------|--------|----------|------|
| 折现率 | 百分比 | 8% | 否 | 用于 NPV 计算 |
| 运营期 | 年 | 20 | 否 | 持有运营年限 |
| 租金年增长率 | 百分比 | 2% | 否 | 租赁收入年增长 |
| 售价年增长率 | 百分比 | 0% | 否 | 销售收入年增长 |
| 销售去化速度 | 下拉选择 | 1.5 万㎡/年 | 否 | 选项：1.0 / 1.2 / 1.5 / 1.8 / 2.0 / 2.5 / 3.0 |
| 租赁去化速度 | 下拉选择 | 1.5 万㎡/年 | 否 | 选项同上 |
| 稳定出租率 | 百分比 | 90% | 否 | 爬坡完成后稳定出租率 |
| 租赁爬坡年数 | 年 | 3 | 否 | 从 0% 爬升到稳定出租率的年数 |
| 是否开启预售 | 是/否 | 否 | 否 | 默认关闭 |
| 建设期第 N 年预售回款比例 | 百分比 | 见说明 | 否 | 开启预售后输入，默认第 1 年 30%、第 2 年 70% |
| 敏感性分析变量范围 | 多选/输入 | 默认 ±10%、±20% | 否 | 用户可自定义 |

## 七、动态模型核心计算逻辑

### 7.1 建设期现金流

| 年份 | 现金流出 | 现金流入 |
|------|----------|----------|
| 0 | 土地款、部分前期/建安 | 无（或预售回款） |
| 1 | 建安支出 | 预售回款（若开启） |
| 2 | 建安尾款 | 预售回款（若开启） |

### 7.2 运营期现金流

| 年份 | 现金流入 | 现金流出 |
|------|----------|----------|
| 第 1 年 | 部分销售收入 + 起步租赁收入 | 运营费用、税金 |
| 第 2 年 | 部分销售收入 + 爬坡租赁收入 | 运营费用、税金 |
| ... | 按去化周期逐步释放 | 运营费用、税金 |
| 稳定期 | 稳定租赁收入 | 运营费用、税金 |

### 7.3 建设期末清算

- 土地增值税：按总销售收入清算。
- 所得税：按利润总额清算。

### 7.4 核心输出指标

| 指标 | 公式 |
|------|------|
| NPV | Σ（第 t 年净现金流 / (1 + 折现率)^t） |
| IRR | 使 NPV = 0 的折现率 |
| 自有资金 IRR | 使自有资金 NPV = 0 的折现率 |
| 动态投资回收期 | 累计折现现金流由负转正的年份 |

## 八、敏感性分析

### 8.1 默认分析变量

| 变量 | 默认波动范围 |
|------|--------------|
| 销售单价 | ±10%、±20% |
| 租金单价 | ±10%、±20% |
| 土地价格 | ±10%、±20% |
| 建安成本 | ±10%、±20% |
| 去化周期 | 1.0~3.0 万㎡/年 |
| 出租率 | 80%~95% |
| 运营费用率 | ±20% |
| 建设周期 | 1~3 年 |
| 融资利率 | 4%~8% |
| 融资占比 | 50%~80% |
| 租金年增长率 | 0%~5% |
| 售价年增长率 | 0%~3% |

### 8.2 输出指标

- NPV
- IRR
- 自有资金 IRR
- 动态投资回收期
