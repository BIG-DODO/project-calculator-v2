# 动态投资分析设计文档

## 一、设计目标

在静态投资分析基础上，引入时间维度，计算项目多年现金流、NPV、IRR、自有资金 IRR、动态投资回收期，并提供敏感性分析。

---

## 二、与其他模块的联动关系

```
产品配置结果 / 综合指标表
        ↓
投资估算表
        ↓
静态投资分析表
        ↓
动态投资分析表
```

- 若用户未完成投资估算，提示「请先进行投资估算」；
- 若用户已完成投资估算但未完成静态投资分析，提示「请先进行静态投资分析」。

## 七、动态模型核心计算逻辑

### 7.1 基本假设

1. **默认不考虑预售**：预售开关默认关闭，但保留开启能力；
2. **预售比例**：预留开关打开时，默认可售部分全部可预售；
3. **建设期末统一清算**：土地增值税、所得税在建设期末统一清算；
4. **全部费用化**：运营期不留融资余额，利息费用化；
5. **不考虑建设期租金收益**：工业项目竣备前通常不能出租；
6. **销售/租赁爬坡**：工业项目去化存在爬坡期，销售/租赁按年去化速度逐步释放。

### 7.2 建设期现金流

| 年份 | 现金流出 | 现金流入 |
|------|----------|----------|
| 0 | 土地款、部分前期/建安 | 无（或预售回款） |
| 1 | 建安支出 | 预售回款（若开启） |
| 2 | 建安尾款 | 预售回款（若开启） |

- 建设期支出按年均匀投入；
- 自有资金支出 = 总投资 × (1 - 融资占比)；
- 融资本金支出 = 总投资 × 融资占比。

### 7.3 运营期现金流

| 年份 | 现金流入 | 现金流出 |
|------|----------|----------|
| 第 1 年 | 部分销售收入 + 起步租赁收入 | 运营费用、税金 |
| 第 2 年 | 部分销售收入 + 爬坡租赁收入 | 运营费用、税金 |
| ... | 按去化周期逐步释放 | 运营费用、税金 |
| 稳定期 | 稳定租赁收入 | 运营费用、税金 |

- 销售收入按年去化速度释放，考虑售价年增长率；
- 租赁收入按年租赁去化速度释放，考虑爬坡期和租金年增长率；
- 运营费用、税金按静态分析规则计算。

### 7.4 融资与利息

- 融资方案与投资估算联动；
- 运营期全部费用化，不留融资余额；
- 利息支出按静态分析中的财务费用口径计入。

### 7.5 建设期末清算

- 土地增值税：按总销售收入清算；
- 所得税：按利润总额清算。

### 7.6 核心输出指标

| 指标 | 公式 |
|------|------|
| NPV | Σ（第 t 年净现金流 / (1 + 折现率)^t） |
| IRR | 使 NPV = 0 的折现率 |
| 自有资金 IRR | 使自有资金 NPV = 0 的折现率 |
| 动态投资回收期 | 累计折现现金流由负转正的年份 |

## 八、敏感性分析

### 8.1 默认分析变量

| 变量 | 默认波动范围 |
|------|--------------|
| 销售单价 | ±10%、±20% |
| 租金单价 | ±10%、±20% |
| 土地价格 | ±10%、±20% |
| 建安成本 | ±10%、±20% |
| 去化周期 | 1.0~3.0 万㎡/年 |
| 出租率 | 80%~95% |
| 运营费用率 | ±20% |
| 建设周期 | 1~3 年 |
| 融资利率 | 4%~8% |
| 融资占比 | 50%~80% |
| 租金年增长率 | 0%~5% |
| 售价年增长率 | 0%~3% |

### 8.2 输出指标

- NPV
- IRR
- 自有资金 IRR
- 动态投资回收期

### 8.3 展示形式

- 敏感性分析表；
- 龙卷风图（按影响程度排序）。

## 十、已确认 / 待细化事项

### 已确认事项

1. 动态模型默认不考虑预售，但预留预售开关；预售比例默认可售部分全部可预售。
2. 销售/租赁去化速度提供选项：1.0 / 1.2 / 1.5 / 1.8 / 2.0 / 2.5 / 3.0 万㎡/年。
3. Excel 表按年展示，页面端只展示关键年份（0、1、2、5、10、20 年）。
4. 建设期末统一清算土地增值税和企业所得税。
5. 全部费用化，运营期不留融资余额。
6. 敏感性分析变量包括：销售单价、租金单价、土地价格、建安成本、去化周期、出租率、运营费用率、建设周期、融资利率、融资占比、租金年增长率、售价年增长率。
7. 增加自有资金 IRR 指标。
8. 资金缺口公式沿用静态分析：`总投 - (1 - 融资占比) × 总投 - 销售收入`。

### 待细化事项

1. 建设期支出在各年的具体分配比例（默认均匀投入，后续可根据付款节点细化）。
2. 预售回款是否影响建设期末的土增税和所得税清算（当前按建设期末统一清算处理）。
3. 敏感性分析 tornado 图是否按影响程度自动排序。
